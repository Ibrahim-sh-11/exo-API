README

Ce projet est un petit site web regroupant l’appel de deux API différentes : CatFacts et PokéAPI.
Le site inclut également une animation visuelle simulant une cyberattaque, ainsi qu’un fond animé de type Matrix.

Fonctionnalités :
CatFacts API
Récupère un fait aléatoire sur les chats depuis l’API publique : https://catfact.ninja/fact

PokéAPI permet d’afficher les informations d’un Pokémon en entrant son nom.
Les données (statistiques et sprite) proviennent de : https://pokeapi.co/

Simulation d’attaque:
Une animation en plein écran imitant une séquence d’attaque informatique.
Il s’agit uniquement d’un effet visuel pour l’interface.
Effets visuels
Le site intègre une animation de pluie de caractères inspirée de Matrix ainsi qu’un effet de brouillard léger pour l’ambiance
